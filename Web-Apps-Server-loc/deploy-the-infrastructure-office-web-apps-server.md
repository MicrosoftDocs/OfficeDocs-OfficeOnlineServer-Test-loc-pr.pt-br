# Implantar a infraestrutura: Servidor do Office Web Apps

 

_<strong>Aplica-se a:</strong>Office Web Apps Server_

_<strong>Tópico modificado em:</strong>2016-12-16_

**Resumo:** contém artigos sobre o Servidor do Office Web Apps e como ele oferece funcionalidade dos Office Web Apps aos hosts.

**Público:** profissionais de TI

Servidor do Office Web Apps é um novo produto de servidor do Office que fornece versões baseadas em navegador do Word, do PowerPoint, do Excel e do OneNote. Um único farm do Office Web Apps pode suportar usuários que acessam arquivos do Office por meio do SharePoint 2013, do Lync Server 2013, do Exchange Server 2013, de pastas compartilhadas e sites. O novo modelo de implantação autônomo significa que você pode gerenciar atualizações ao seu farm do Servidor do Office Web Apps independentemente de outros produtos do Office Server implantados em sua organização.
