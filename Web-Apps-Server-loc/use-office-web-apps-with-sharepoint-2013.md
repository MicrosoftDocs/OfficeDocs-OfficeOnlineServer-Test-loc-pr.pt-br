# Usar o Office Web Apps com o SharePoint 2013

 

_**Aplica-se a:**Office Web Apps, SharePoint Foundation 2013, SharePoint Server 2013_

_**Tópico modificado em:**2017-05-26_

**Resumo:** Contém artigos que lhe ajudam a aprender como implantar e configurar o Office Web Apps para SharePoint 2013 local.

**Audiência:** profissionais de TI

Quando usado com SharePoint 2013 local, Office Web Apps fornece versões atualizadas do Word Web App, Excel Web App, PowerPoint Web App e OneNote Web App. Os usuários podem exibir e, opcionalmente, editar documentos Office usando um navegador da web com suporte em computadores e em diferentes dispositivos móveis, como Windows Phones, iPhones e iPads. Além dos novos recursos no Office Web Apps, os métodos de implantação e arquitetura também alterou, que permitem Office Web Apps fornecer funcionalidade de visualização de Lync Server 2013.

![Logotipo do Office 2013](images/JJ219456.a106e261-2cd0-43b7-af77-92de7e4b6fb9(Office.15).png "Logotipo do Office 2013")Os artigos na tabela a seguir são destinados aos profissionais de TI que planejam configurar o SharePoint 2013 a fim de usar o Office Web Apps no local em suas organizações.
